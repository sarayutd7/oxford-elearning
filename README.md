# 📚 Oxford CEFR Vocabulary JSON

ชุดคำศัพท์ภาษาอังกฤษระดับ CEFR (A1 / A2 / B1 / B2 ฯลฯ) ในรูปแบบ JSON พร้อมข้อมูลครบสำหรับนำไปใช้สร้าง:

- Vocabulary App
- Flashcards
- Quiz System
- Language Learning Platform
- AI / NLP Project
- Mobile App
- Web App

---

# ✨ Features

แต่ละคำศัพท์มีข้อมูลครบ:

- ✅ English meaning
- ✅ Thai meaning
- ✅ IPA pronunciation
- ✅ Thai pronunciation
- ✅ Part of speech
- ✅ Example sentences
- ✅ Synonyms
- ✅ Tags
- ✅ CEFR level

---
